# Collections in JAVA
It is group of inbuilt classes and interface to do
operations.

**JAVA COLLECTION FRAMEWORKS**
1. ArrayList,LinkedList->List
2. HashSet,TreeSet->Set
3. PriorityQueue,ArrayDeque->Queue
4. HashMap,TreeMap->Map

**Algorithms**
1.sort()
2.reverse()
3.max(),min()
4.binarySearch()


SampleInput
5
john tom
john mary
john tom
mary anna
mary anna
Sample Output
1
2
2
3
3
Explanation
Set={(john tom)}->1
Set={(john tom),(john mary)}->2
Set={(john tom),(john mary)}->2
Set={(john tom),(john mary),(marry anna)}-3
Set={(john tom),(john mary),(marry anna)}-3
